# DRO Lathe - Android App

Цифровая индикация для токарного станка с подключением к STM32 через Bluetooth (HC-06).

## Функции

- **4 строки координат**: X (радиус), D (диаметр), Z (продольная), L (длина)
- **4 инструмента** с сохранением смещений
- **Засечки** с сигналом приближения
- **Помощник резьбы** (метрическая, дюймовая, трубная)
- **Измерение углов** по двум точкам
- **Настройки**: разрешение энкодера (1/5 мкм), цвета, ориентация

## Требования

- Android 4.2+ (API 17)
- Bluetooth
- STM32 с HC-06

## Формат данных от STM32

```
ValX|ValZ\n
```

Пример: `1234|5678\n`

Значения в единицах энкодера. Реальное положение = Val × Resolution.

## Сборка в Android Studio

1. Откройте Android Studio
2. File → Open → выберите папку `DRO`
3. Дождитесь синхронизации Gradle
4. Build → Build APK(s)
5. APK будет в `app/build/outputs/apk/debug/`

## Подключение к HC-06

1. Сопрягите HC-06 с планшетом в настройках Bluetooth
   - Обычно пароль: `1234` или `0000`
2. Запустите приложение
3. Нажмите "Подключить"
4. Выберите HC-06 из списка

## Разрешения

Приложение запрашивает:
- `BLUETOOTH` / `BLUETOOTH_ADMIN` - для подключения
- `BLUETOOTH_CONNECT` / `BLUETOOTH_SCAN` - Android 12+
- `ACCESS_FINE_LOCATION` - требуется для сканирования Bluetooth

## Использование

### Обнуление координат
- Нажмите **Zero** рядом с X или Z
- D и L пересчитываются автоматически (D = 2×X, L = Z)

### Инструменты
- Короткое нажатие - выбор инструмента (применяет смещение)
- Длинное нажатие - редактирование смещения

### Засечки
- Добавьте засечку для сигнала при приближении к заданной позиции
- Укажите ось (X или Z) и позицию

### Помощник резьбы
- Выберите тип резьбы и шаг
- Приложение покажет глубину резьбы и рекомендуемое число проходов
